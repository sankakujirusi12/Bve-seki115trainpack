# 概要
Bve5.8/6用 金サワ・近キト・岡オカ・広ヒロ・広セキ113/115/117/415系車両パックです。車両指定は連絡なしで自由に行えます。  

# ファイル構成  
```
Scenarios - △12△  
　　　　 - Rock_On  
　　　　 - GeneralAtsPlugin  
```
となるよう配置してください。  

# 導入が必要なもの
JR線車両パック(Rock_On様)
